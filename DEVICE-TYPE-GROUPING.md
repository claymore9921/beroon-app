# Manager scooter grouping

The manager scooter list now groups scooters by device type.

- Each device type has a separate section.
- The section header shows the device type and scooter count.
- Scooters inside each group are sorted by plate.
- Scooters without a device type appear under `بدون نوع دستگاه`.
- Existing status filters and device information remain unchanged.
- The Excel XML entity decoding bug is also fixed in this build.
