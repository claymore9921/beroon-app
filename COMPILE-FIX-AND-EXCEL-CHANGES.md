# اصلاح خطای کامپایل و خروجی Excel

- حذف استفاده اشتباه از متغیر تعریف‌نشده `params` در `send_scooter_to_workshop/2`.
- رفع هشدار charlist در `Beroon.Calendar` با استفاده از `~c""`.
- نمایش همه نوع دستگاه‌های ثبت‌شده در خروجی، حتی نوع‌های غیرفعال.
- افزودن ستون‌های داخل تعمیرگاه، در انتظار قطعه، انبار نو و فروش‌رفته.
- افزودن جمع هر ردیف، جمع هر ستون و جمع کل همه سلول‌های آماری.
